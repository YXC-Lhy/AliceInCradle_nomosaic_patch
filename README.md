## 自动全版本补丁安装工具（推荐） 
适配所有旧版本和未来新版本
下载工具，放到游戏主程序AliceInCradle.exe所在的目录运行（放在游戏目录可自动识别，也可以放在任意位置手动选择游戏目录），选择安装补丁，等待一分钟即可安装成功
